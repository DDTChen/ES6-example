2. Class
   雖然javascript本來就可以用prototype做到物件導向的設計，ES6的class讓這件事更好寫。
   但是這個class的功能並不多像：只能宣告成員方法，無法宣告成員屬性，只有static、public，沒有private
   https://babeljs.io/docs/learn-es2015/#classes
